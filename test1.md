- clone : 원격 저장소를 로컬 컴퓨터에 복사
- add : 스테이지 영역에 작업 파일 추가
- commit : 스테이지 영역에 파일들을 가지고 커밋을 만듬
- push : 원격 저장소에 커밋을 업로드

- **브랜치(branch): 독립적인 작업 공간**
- **머지(merge): 브랜치 간 작업 통합**
- **체크아웃(checkout): 특정 브랜치 또는 커밋으로 이동**
